# music-app
   Vue2.x 音乐 App
  **<font size=6>简介</font>**
  
  <a href="https://cn.vuejs.org/" target="_blank">官方入门教程</a>

  **<font size=6>安装</font>**
  
  下载项目后，进入项目文件夹所在的终端

```javascript
  npm install
```
**<font size=6>打包</font>**
  
  下载项目后，进入项目文件夹所在的终端

```javascript
  npm run build
```
  
  **<font size=6>启动</font>**

  ```javascript
    npm run dev
  ```
  
  最后浏览器打开http://localhost:8080
